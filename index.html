A propos de moi


Ambitieuse & rigoureuse

Stimulée par la complexité logique des projets web, je mets ma polyvalence au service de ce domaine. Combinant une réflexion critique solide et une curiosité constante face à l'évolution continue des connaissances de ce métier, je défends fermement l'exigence d'innovation pour garantir des solutions pleinement adaptées à vos besoins.

Pourquoi ce métier ?

La polyvalence est la clé de voûte par laquelle je reste stimulée intellectuellement. Ce fait est l’une des raisons pour lesquelles je souhaite faire carrière dans ce domaine. De plus, la dimension systémique des langages de programmation me plaît car c’est un métier théorique dont sa pratique et son actualisation constantes sont intrinsèquement liées. La dimension évolutive est donc un défi pour lequel je suis enthousiaste et il est important pour moi d’être dans une dynamique d’apprentissage continu. Vous me direz que cela est rendu possible dans différents secteurs d’activités. Cependant, la croissance exponentielle des technologies est ce qui attise fortement ma curiosité. Enfin, le front-end permet le déploiement de ma créativité et cela représente un véritable atout. Ce métier représente pour moi l’établissement de tableaux (dans son sens artistique) rendus fonctionnels.

Mes projets

Agence de voyage

Langages : HTML / CSS / JS

J'ai conceptualisé un site web statique en imaginant avoir été missionnée par une agence de tourisme. Pour donner un ancrage professionnel à mes compétences, il est primordial de me donner des défis.

Ce site est responsive. Sa palette de couleurs est en référence à celles de l'île, et les clichés photographiques ont été pris par moi. Sa puissance marketing réside dans la combinaison d'un aspect informatif suscitant l'envie chez le lecteur pour aboutir à la proposition de circuits. Le fil conducteur de ce site repose sur la création d'un besoin client fondé sur le désir.

Freshly Restaurant

Langages : HTML / CSS

Ce site vitrine a été conçu pour valoriser un restaurant en mettant en avant son identité culinaire et son ambiance chaleureuse. Composé de quatre pages élégamment structurées, il offre une présentation captivante du restaurant, une mise en avant soignée des menus, et une page dédiée aux contacts intégrant une carte interactive. L’architecture du site a été pensée pour maximiser l'expérience utilisateur et renforcer la visibilité du restaurant en ligne. L’ensemble est optimisé pour être totalement responsive.

GitHub User Finder

Langages : HTML / CSS / JavaScript

Ce projet est une application web interactive permettant de rechercher les informations d’un utilisateur GitHub en temps réel. Grâce à une interface intuitive, elle affiche le nom, l'avatar, le nombre de dépôts publics, ainsi que la date de création du compte et la durée depuis cette création. L'application gère également les erreurs de saisie ou d'utilisateur inexistant et fournit des notifications appropriées.. Elle illustre une approche efficace pour consommer des API de manière asynchrone.
